# 🏠 마스터 옵시디언 — 모두를 위한 완벽 가이드

> [!tip] 환영합니다
> 이 보관소(Vault)는 옵시디언을 처음 시작하는 분을 위한 한국어 시작 키트입니다.
> 옵시디언에서 이 폴더를 **보관소로 열기(Open folder as vault)** 하면 바로 사용할 수 있어요.

---

## 1. 옵시디언이란?

옵시디언은 **내 컴퓨터에 저장되는 마크다운(Markdown) 메모 앱**입니다.

- 📂 모든 노트는 그냥 `.md` 텍스트 파일 → 평생 내 것, 잠기지 않음
- 🔗 노트끼리 `[[링크]]`로 연결 → 제2의 뇌(Second Brain)
- 🧩 플러그인으로 무한 확장
- 💻 윈도우 · 맥 · 리눅스 · 안드로이드 · iOS 모두 지원

---

## 2. 가장 먼저 익힐 3가지

| 기능 | 방법 | 설명 |
|---|---|---|
| 새 노트 | `Ctrl/Cmd + N` | 노트 하나 = 생각 하나 |
| 노트 연결 | 본문에 `[[` 입력 | 다른 노트 이름을 입력해 연결 |
| 빠른 검색 | `Ctrl/Cmd + O` | 모든 노트로 순간 이동 |

## 3. 이 보관소 폴더 구조

```
옵시디언-보관소/
├── 00-시작      ← 지금 이 가이드
├── 10-노트      ← 평소 메모를 여기에
├── 20-템플릿    ← 반복 양식 (일일노트 등)
├── 30-자료      ← 참고 자료/스크랩
└── 90-보관      ← 다 쓴 노트 보관
```
