## ✅ 미완료 할 일 (전체 노트에서 자동 수집)

```dataview
TASK
WHERE !completed
GROUP BY file.link
SORT file.mtime DESC
LIMIT 20
```

---

## 🕒 최근에 수정한 노트 10개

```dataview
TABLE WITHOUT ID
  file.link AS "노트",
  dateformat(file.mtime, "MM-dd HH:mm") AS "수정"
FROM "10-노트" OR "30-자료"
SORT file.mtime DESC
LIMIT 10
```

---

## 📅 이번 주 일일 노트

```dataview
TABLE WITHOUT ID file.link AS "날짜"
FROM "10-노트/일일노트"
SORT file.name DESC
LIMIT 7
```

---

## 🏷️ 많이 쓰는 태그

```dataview
TABLE WITHOUT ID rows.file.link AS "노트"
FROM ""
FLATTEN file.tags AS 태그
WHERE 태그
GROUP BY 태그
SORT length(rows) DESC
LIMIT 10
```
