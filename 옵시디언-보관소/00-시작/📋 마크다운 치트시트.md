# 📋 마크다운 치트시트

옵시디언 노트는 마크다운으로 작성합니다. 자주 쓰는 문법만 모았습니다.

## 글자 꾸미기
```markdown
**굵게**      → 굵게
*기울임*      → 기울임
~~취소선~~    → 취소선
==형광펜==    → 형광펜
`코드`        → 코드
```

## 제목
```markdown
# 제목1
## 제목2
### 제목3
```

## 목록 / 할 일
```markdown
- 점 목록
1. 번호 목록
- [ ] 할 일 (미완료)
- [x] 할 일 (완료)
```

## 링크 / 임베드 (옵시디언 핵심)
```markdown
[[노트 이름]]            ← 다른 노트로 링크
[[노트 이름|보이는 글자]]  ← 별명으로 링크
[[노트 이름#제목2]]       ← 노트 안 특정 제목으로
![[노트 이름]]           ← 다른 노트 내용을 끼워넣기(임베드)
![[그림.png]]            ← 이미지 삽입
[구글](https://google.com) ← 웹 링크
```

## 인용 / 콜아웃
```markdown
> 인용문

> [!tip] 팁 제목
> 강조 박스 (note, tip, warning, quote, info, example 등)
```

## 표
```markdown
| 항목 | 값 |
|---|---|
| 가 | 1 |
| 나 | 2 |
```

## 구분선 / 코드 블록
````markdown
---

```python
print("코드 블록")
```
````

## 태그 / 각주
```markdown
#태그   #프로젝트/하위태그

본문[^1]
[^1]: 각주 내용
```

> [!tip]
> 편집/미리보기 전환은 `Ctrl/Cmd + E`. 라이브 프리뷰 모드면 입력과 동시에 결과가 보입니다.
