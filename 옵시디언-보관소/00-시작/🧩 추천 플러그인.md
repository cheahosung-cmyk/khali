# 🧩 추천 플러그인

> [!info] 설치 방법
> `설정(⚙️) → 커뮤니티 플러그인 → 제한 모드 끄기 → 검색 후 설치 → 활성화`

> [!tip] 이 보관소는 미리 추천해 둠
> `community-plugins.json`에 **Dataview**, **Calendar**가 등록돼 있어, 제한 모드만 끄고 설치하면 [[🏠 마스터 옵시디언 - 시작하기]] 대시보드가 바로 작동합니다. 자세한 건 [[🔍 Dataview 활용법]].

## 처음엔 이것만 (입문)

| 플러그인 | 무엇을 해주나 |
|---|---|
| **Dataview** | 노트를 자동 표/할 일 목록으로 (홈 대시보드 핵심) → [[🔍 Dataview 활용법]] |
| **Calendar** | 달력에서 클릭해 일일 노트 작성 |
| **Templates** (기본 내장) | 반복 양식 자동 삽입 |
| **Outline** (기본 내장) | 긴 문서의 목차 사이드바 |

## 익숙해지면 (확장)

| 플러그인 | 무엇을 해주나 |
|---|---|
| **Dataview** | 노트를 데이터베이스처럼 자동 표/목록화 |
| **Templater** | 날짜·변수가 들어가는 고급 템플릿 |
| **Excalidraw** | 노트 안에서 손그림·다이어그램 |
| **Tasks** | 할 일 관리 (마감일, 반복) |
| **Style Settings** | 테마/CSS를 메뉴로 손쉽게 조절 |

## 기본 내장 플러그인 (켜기만 하면 됨)

`설정 → 핵심 플러그인`에서 켭니다.
- **일일 노트(Daily notes)** — 날짜별 노트 자동 생성
- **그래프 뷰(Graph view)** — 노트 연결망 시각화
- **백링크(Backlinks)** — 이 노트를 언급한 다른 노트들
- **빠른 전환(Quick switcher)** — `Ctrl/Cmd + O`

> [!warning]
> 플러그인은 한 번에 많이 깔지 마세요. **1~2개씩** 늘려가며 익히는 게 좋습니다.
