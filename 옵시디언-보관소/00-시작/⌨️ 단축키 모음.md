# ⌨️ 단축키 모음

> [!info] 표기 안내
> `Ctrl` = 윈도우/리눅스, `Cmd(⌘)` = 맥. 둘 다 적용되면 `Ctrl/Cmd`로 표기합니다.

## 필수 단축키

| 동작 | 단축키 |
|---|---|
| 새 노트 만들기 | `Ctrl/Cmd + N` |
| 빠른 전환 (노트 검색·이동) | `Ctrl/Cmd + O` |
| 전체 검색 | `Ctrl/Cmd + Shift + F` |
| 명령어 팔레트 (모든 기능) | `Ctrl/Cmd + P` |
| 현재 폴더에 노트 | `Ctrl/Cmd + N` |
| 그래프 뷰 열기 | 명령어 팔레트 → "그래프" |

## 편집

| 동작 | 단축키 |
|---|---|
| 굵게 (**볼드**) | `Ctrl/Cmd + B` |
| 기울임 (*이탤릭*) | `Ctrl/Cmd + I` |
| 미리보기 ↔ 편집 모드 전환 | `Ctrl/Cmd + E` |
| 링크 삽입 `[[ ]]` | `[[` 입력 |
| 체크박스 토글 | `Ctrl/Cmd + L` |
| 줄 복제 | `Ctrl/Cmd + Shift + D` |

## 창/탭

| 동작 | 단축키 |
|---|---|
| 오른쪽으로 분할 | `Ctrl/Cmd + \` |
| 탭 닫기 | `Ctrl/Cmd + W` |
| 뒤로 / 앞으로 이동 | `Alt + ← / →` |
| 사이드바 토글 | `Ctrl/Cmd + Shift + 좌/우` 클릭 |

> [!tip]
> 단축키가 기억나지 않으면 무조건 `Ctrl/Cmd + P` (명령어 팔레트). 거의 모든 기능을 이름으로 찾을 수 있습니다.
