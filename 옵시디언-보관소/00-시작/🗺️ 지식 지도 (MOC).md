# 🗺️ 지식 지도 (MOC, Map of Content)

> [!info] MOC란?
> **Map of Content** = 관련 노트들을 한곳에 모아 링크해 두는 '목차 노트'입니다.
> 폴더보다 유연하게, 주제별로 노트를 묶을 수 있어 옵시디언의 핵심 활용법입니다.

## 사용법
1. 주제가 생기면 이 목록에 `[[주제 노트]]`를 추가합니다.
2. 주제 노트 안에서 다시 세부 노트들을 링크합니다.
3. 그래프 뷰(`Ctrl/Cmd + P` → 그래프)로 연결망을 확인합니다.

## 🔗 자동 색인 (Dataview)

아래는 `10-노트/주제` 폴더의 모든 주제 노트를 자동으로 모읍니다.

```dataview
LIST
FROM "10-노트/주제"
SORT file.name ASC
```

> [!tip]
> 새 주제 노트는 `10-노트/주제` 폴더에 만들면 위 목록에 자동으로 나타납니다.
