# 옵시디언 보관소 (Master Obsidian 시작 키트)

한국어 옵시디언 입문용 보관소(Vault)입니다.

## 사용 방법

1. [옵시디언 다운로드](https://obsidian.md) 후 설치
2. 옵시디언 실행 → **보관소 열기 → 폴더를 보관소로 열기(Open folder as vault)**
3. 이 `옵시디언-보관소` 폴더를 선택
4. `00-시작/🏠 마스터 옵시디언 - 시작하기` 노트부터 읽기

## 구성

| 폴더 | 용도 |
|---|---|
| `00-시작` | 시작 가이드, 단축키, 워크플로우, 플러그인, CSS |
| `10-노트` | 평소 메모 |
| `20-템플릿` | 일일/독서 노트 양식 |
| `30-자료` | 첨부파일·스크랩 |
| `90-보관` | 끝난 노트 보관 |

> 모든 문서는 일반 마크다운(`.md`)이라 옵시디언이 없어도 어떤 편집기로든 읽을 수 있습니다.
