# 옵시디언 보관소 (Master Obsidian 시작 키트)

한국어 옵시디언 입문·활용 보관소(Vault)입니다. 대시보드·템플릿·가이드가 미리 갖춰져 있어 열자마자 바로 쓸 수 있습니다.

## 사용 방법

1. [옵시디언 다운로드](https://obsidian.md) 후 설치
2. 옵시디언 실행 → **보관소 열기 → 폴더를 보관소로 열기(Open folder as vault)**
3. 이 `옵시디언-보관소` 폴더 선택
4. `00-시작/🏠 마스터 옵시디언 - 시작하기` 노트(홈 대시보드)부터 읽기
5. 대시보드를 채우려면 `설정 → 커뮤니티 플러그인 → 제한 모드 끄기` 후 **Dataview**, **Calendar** 설치

## 구성

| 폴더 | 용도 |
|---|---|
| `00-시작` | 홈 대시보드, 단축키, 워크플로우, 플러그인, Dataview, 마크다운 치트시트, MOC, FAQ, CSS |
| `10-노트` | 평소 메모 · `일일노트/` · `주제/`(MOC용) · 받은 편지함(Inbox) |
| `20-템플릿` | 일일 · 독서 · 회의 · 프로젝트 · 주간 리뷰 템플릿 |
| `30-자료` | 첨부파일·스크랩 (이미지 드롭 시 여기로) |
| `90-보관` | 끝난 노트 보관 |

## 미리 된 설정 (`.obsidian`)
- 일일 노트 → `10-노트/일일노트` 폴더 + 일일 템플릿 자동 연결
- 템플릿 폴더 → `20-템플릿`
- 첨부파일 → `30-자료`
- 추천 플러그인(Dataview·Calendar) 등록, 기본 폰트 17px

> 모든 문서는 일반 마크다운(`.md`)이라 옵시디언이 없어도 어떤 편집기로든 읽을 수 있습니다.
