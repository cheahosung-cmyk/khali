# 웰스타임 시설 관리 우선순위 체크리스트

관리자가 직접 정리한 우선순위에, 인수인계 녹음에서 나온 누락 항목을 보완한 버전.

## 🔴 1순위 (긴급)

- [ ] **수신반 배터리 교체** (네모난 것, 오래가는 좋은 것으로)
- [ ] **전기 계약전력 변경: 23kW → 5kW**
  - 문의: 한전 ☎123 (휴대폰은 032-123) — 고객번호 준비하고 전화
  - 서류: 인감증명(사진 가능) + 도장, 개인/법인 여부 확인 → 메일로 받아 회신
  - 하루 늦을수록 손해 (월 약 12만원 이상 절약)

## 🟠 2순위 (행정 업무)

- [ ] **사업자 등록 확인** — 세무사무실 또는 세무서 문의
  - 상가관리업 · 주차장 요금 수입
  - 기존 임대사업자에 업종 추가 가능한지 / 별도 등록 필요한지 확인 후 정리
- [ ] **업체 세금계산서 명의변경** (사업자 정리 후 바로)
  - 소방·전기·엘리베이터·인터넷 업체에 사업자번호 전달
  - 부가세는 일반관리비에만 (수도세·전력기금 제외, 나눠 발행)
- [ ] **업체 미수금 확인** — 명의변경 전에 필수 (미수 있으면 변경 안 해줌)
  - 소방설비 미납 잔액 70~80만원 언급됨

## 🟡 3순위 (시설 관리)

- [ ] **1층 엘리베이터 옆 소방안전관리자 연락처 변경** (소방관이 이 번호로 전화함)
- [ ] **업체 연락처 정리** — 소방 / 전기 / 엘리베이터
  - 담당자 번호 + 회사(비상) 번호 둘 다 확보, 1층 게시
  - 소방: 우슬방제 010-5098-1379 (과장 개인번호 — 회사번호 별도 확보)
- [ ] **CCTV 복구** — KT텔레캅 ☎1588-0112 (5년 만료로 현재 작동 안 함, 유료 복구)
- [ ] **스터디카페 공용전기 정산 반영** — 다음 달 관리비부터 차감/부과

## 🟢 4순위 (교육·기타)

- [ ] **소방안전관리자 2급 교육 일정 확인 및 신청** (매달 있음, 7월 명의이전 전에)
- [ ] 관리비/주차요금 통장 분리
- [ ] 인터넷 TV 해지 (4만원 → 1.7만원, 인터넷만 유지)

## 📅 10일 — 이전 관리자 인수인계 날

- [ ] CCTV 아이디·비번 인수
- [ ] 주차기 등록정보 초기화 후 재입력
- [ ] 관리비 세금계산서 발급 이력 확인
- [ ] 업체 미수금 정리
