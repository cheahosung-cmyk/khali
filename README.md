# khali

상가건물 관리소장을 위한 AI 직원 50명 관리 시스템.

시설관리·안전소방·관리비회계·임대차·민원·법무 등 관리소장의 실제 직무를 10개 부서로
나누고, 부서마다 5명씩 총 50명의 AI 직원(역할·시스템 프롬프트·권장 모델)을 정의한다.
로스터는 CLI와 API로 조회하고, Claude Code 서브에이전트 파일로 내보내 바로 쓸 수 있다.

## 부서 구성 (10개 부서 × 5명)

| 부서 | 직원 |
|------|------|
| 시설관리 | 전기설비, 소방설비, 승강기, 냉난방·공조, 급배수·배관 |
| 안전·소방·보안 | 안전점검 총괄, 소방 훈련 설계, 경비·보안 기획, CCTV·출입, 재난 대응 |
| 미화·환경 | 미화 감독, 폐기물·재활용, 방역·위생, 조경·외관, 에너지 절감 |
| 관리비·회계 | 관리비 부과, 회계·정산, 미납 관리, 예산 편성, 수선충당금 |
| 임대차·입점 | 계약 관리, 공실 마케팅, 시세 분석, 입·퇴점 절차, 상가임대차보호법 자문 |
| 민원·소통 | 민원 응대, 공지문, 상인회 회의 지원, 분쟁 조정, 안내방송·문자 |
| 계약·법무 | 용역계약 검토, 집합건물법 자문, 관리규약, 하자·배상 대응, 보험 관리 |
| 공사·수선 | 수선공사 기획, 견적 비교, 공사 감독·검수, 장기수선계획, 하자보수 청구 |
| 행정·문서 | 관리일지, 법정검사 일정, 문서·대장, 직원 근태·교대, 보고서 작성 |
| 경영지원 | 관리단 집회 지원, 주차 운영, 부대수익 기획, 스마트빌딩 자문, 관리소장 비서 |

## 설치

```bash
pip install -e .
```

## 사용법

```bash
# 직원 50명 목록 보기
khali list

# 부서·키워드로 찾기
khali list --department facilities
khali list --q 승강기

# 직원 상세(시스템 프롬프트 포함) 보기
khali show condo-law-advisor

# Claude Code 서브에이전트로 내보내기 (.claude/agents/*.md 50개 생성)
khali export

# 조회 API 서버 실행 (http://127.0.0.1:8000/docs)
khali serve
```

`khali export`를 프로젝트 루트에서 실행하면 Claude Code에서 각 직원을 서브에이전트로
바로 호출할 수 있다.

## 한계

- AI 직원의 법령(집합건물법, 상가건물 임대차보호법, 소방시설법 등) 관련 답변은
  참고용이며, 분쟁·소송·과태료가 걸린 사안은 반드시 해당 분야 전문가 확인이 필요하다.
- 로스터는 정적 데이터(`src/roster.py`)가 단일 소스다. 직원 추가·수정은 해당 파일을
  고치면 CLI·API·내보내기에 모두 반영된다. DB 기반 CRUD는 아직 없다.
