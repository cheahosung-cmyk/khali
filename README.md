# khali — 관리비 정산 에이전트

관리소장을 위한 **공동주택 관리비 정산 자동화 도구**입니다.
세대 정보·비용 항목·계량기 검침값(엑셀/CSV)만 넣으면, 항목별 배분 규칙에 따라
**세대별 고지서를 자동으로 계산**하고 합계가 정확히 맞는지까지 검증해 줍니다.

## 무엇을 해결하나요?

매달 반복되는 관리비 정산을, 엑셀 수식과 씨름하는 대신 명령 한 줄로 끝냅니다.

- ✅ 항목별 배분 방식 자동 적용 — **면적 비례 / 세대 균등 / 입주 세대 균등 / 사용량(검침) 비례**
- ✅ **원(₩) 단위 잔여까지 정확히 배분** → "세대 합계 = 건물 총액"이 1원도 안 틀림 (최대잉여법)
- ✅ **공실 자동 처리** — 입주 세대만 부담하는 항목/사용량 0 세대는 자동 0원
- ✅ 결과를 **CSV로 저장** → 엑셀에서 바로 열어 고지서 출력
- ✅ 별도 설치 없이 표준 라이브러리만으로 실행 (Python 3.9+)

## 빠른 시작

### 1) 샘플 입력 파일 만들기

```bash
python -m src.cli sample --dir ./sample
```

`sample/` 폴더에 `units.csv`, `costs.csv`, `meters.csv` 예시 파일이 생성됩니다.

### 2) 정산 실행

```bash
python -m src.cli settle \
  --units  ./sample/units.csv \
  --costs  ./sample/costs.csv \
  --meters ./sample/meters.csv \
  --out    ./sample/bills.csv
```

요약이 화면에 출력되고, 세대별 고지서가 `bills.csv`로 저장됩니다.

> `pip install -e .` 로 설치하면 `python -m src.cli` 대신 `khali` 명령을 바로 쓸 수 있습니다.

## 입력 파일 형식 (엑셀에서 CSV로 저장)

### `units.csv` — 세대 정보

| 동 | 호 | 전용면적 | 입주여부 |
|----|----|---------|---------|
| 101 | 1501 | 84.97 | Y |
| 101 | 1602 | 59.95 | N |

- `입주여부`: `Y`/`N` (또는 입주/공실). 공실은 '입주균등' 항목과 사용량 항목에서 자동 제외됩니다.

### `costs.csv` — 비용 항목 (건물 전체 월 총액)

| 항목 | 총액 | 배분방식 |
|------|------|---------|
| 일반관리비 | 3500000 | 면적 |
| 청소비 | 1200000 | 세대균등 |
| 승강기유지비 | 800000 | 입주균등 |
| 세대전기료 | 1280000 | 사용량 |
| 장기수선충당금 | 2000000 | 면적 |

**배분방식** 값:

| 값 | 의미 | 가중치 |
|----|------|--------|
| `면적` | 전용면적 비례 | 전용면적 |
| `세대균등` | 전 세대 균등 | 동일 |
| `입주균등` | 입주 세대만 균등 (공실 제외) | 입주 세대 동일 |
| `사용량` | 검침 사용량 비례 | `meters.csv`의 사용량 |

### `meters.csv` — 검침값 (`사용량` 항목이 있을 때만)

| 동 | 호 | 항목 | 사용량 |
|----|----|------|-------|
| 101 | 1501 | 세대전기료 | 320 |
| 101 | 1501 | 수도료 | 18 |

- `항목`은 `costs.csv`의 항목명과 **정확히 일치**해야 합니다.

## 출력 (`bills.csv`)

세대별로 항목 금액과 합계가 한 줄씩, 맨 아래에 전체 합계 행이 들어갑니다.

```
동,호,전용면적,입주여부,일반관리비,청소비,...,합계
101,1501,84.97,Y,1026066,300000,...,2915883
...
합계,,,,3500000,1200000,...,9860000
```
