# khali — 상가건물 관리비 정산 에이전트

**본인 소유 상가건물**의 건물주(관리소장)를 위한 **관리비 정산·청구 자동화 도구**입니다.
호실 정보·비용 항목·계량기 검침값(엑셀/CSV)만 넣으면, 항목별 배분 규칙에 따라
**호실별 고지서를 자동 산출**하고 합계 검증까지 해 줍니다.

> 웰스타임 빌딩(총 9호실·1,006평)의 실제 정산 구조를 기본 예시로 제공합니다.

## 무엇을 해결하나요?

매달 반복되는 상가 관리비 정산을, 엑셀 수식과 씨름하는 대신 명령 한 줄로 끝냅니다.

- ✅ 항목별 배분 방식 자동 적용 — **면적(평수) 비례 / 호실 균등 / 사용량(검침) 비례**
- ✅ **평당 단가 항목 지원** — 일반관리비처럼 `평수 × 4,000원` 정액 산정
- ✅ **평수 정확비례 배분** — 기존 정산표의 반올림 분배율(0.044 등)로 인한 오차를 바로잡음
- ✅ **원(₩) 단위 잔여까지 정확히 배분** → "항목별 합계 = 총액"이 1원도 안 틀림 (최대잉여법)
- ✅ **공실 자동 분리** — 공실 호실 면적분은 건물주 부담으로 표시(임차인 비전가)
- ✅ **부가세 선택** — 기본 미적용(기존 정산표와 동일), `--vat-rate 0.1`로 켜기 가능
- ✅ 결과를 **CSV로 저장** → 엑셀에서 바로 열어 고지서 작성
- ✅ 별도 설치 없이 표준 라이브러리만으로 실행 (Python 3.9+)

## 빠른 시작

### 1) 예시 입력 파일 만들기 (웰스타임 빌딩 기준)

```bash
python -m src.cli sample --dir ./sample
```

`sample/` 폴더에 `units.csv`, `costs.csv`, `meters.csv`가 생성됩니다.
매달 이 파일의 **숫자만 바꿔서** 정산하면 됩니다.

### 2) 정산 실행

```bash
python -m src.cli settle \
  --units  ./sample/units.csv \
  --costs  ./sample/costs.csv \
  --meters ./sample/meters.csv \
  --out    ./sample/bills.csv
```

요약이 화면에 출력되고, 호실별 고지서가 `bills.csv`로 저장됩니다.

> `pip install -e .` 로 설치하면 `python -m src.cli` 대신 `khali` 명령을 바로 쓸 수 있습니다.

### 3) 카카오톡 발송용 고지 텍스트 만들기

```bash
python -m src.cli notice \
  --units  ./sample/units.csv \
  --costs  ./sample/costs.csv \
  --meters ./sample/meters.csv \
  --out    ./sample/notices.txt \
  --title  "웰스타임 빌딩 2024년 12월 관리비" \
  --account "국민은행 123456-78-901234 (예금주: 홍길동)" \
  --due    "2025년 1월 10일까지"
```

`notices.txt`에 호실별 안내문이 구분선과 함께 생성됩니다. 각 블록을 복사해 카카오톡/문자로 보내면 됩니다. 예:

```
━━━━━━━━━ (1/9) 원유로 ━━━━━━━━━
[웰스타임 빌딩 2024년 12월 관리비]
원유로 (44평)

· 일반관리비 : 176,000원
· 공동전기료(기본) : 8,891원
· 공동전기료(전력량) : 24,907원
· 공동전기료(공용) : 982원
· 공동수도료(세대) : 32,576원
· 공동수도료(공용) : 13,730원

▶ 합계 : 257,086원

입금계좌: 국민은행 123456-78-901234 (예금주: 홍길동)
납부기한: 2025년 1월 10일까지
감사합니다.
```

## 입력 파일 형식 (엑셀에서 CSV로 저장)

### `units.csv` — 호실 정보

| 호실 | 평수 | 입주여부 |
|------|------|---------|
| 원유로 | 44 | Y |
| 드림스터디 | 207 | Y |

- `입주여부`: `Y`/`N` (또는 임차/공실). 비우면 임차로 간주합니다.

### `costs.csv` — 비용 항목

| 항목 | 총액 | 배분방식 | 단가 |
|------|------|---------|------|
| 일반관리비 |  | 면적 | 4000 |
| 공동전기료(기본) | 203280 | 면적 |  |
| 공동전기료(전력량) | 569470 | 면적 |  |
| 공동전기료(공용) | 22440 | 면적 |  |
| 공동수도료(세대) | 417566 | 사용량 |  |
| 공동수도료(공용) | 313914 | 면적 |  |

- **단가**가 있으면 `평수 × 단가`로 정액 산정하고 총액은 자동 계산됩니다(일반관리비).
- 단가가 없으면 **총액**을 배분방식에 따라 나눕니다.

**배분방식** 값:

| 값 | 의미 | 가중치 |
|----|------|--------|
| `면적` | 분양평수 비례 | 평수 |
| `호실균등` | 전 호실 균등 | 동일 |
| `사용량` | 검침 사용량 비례 | `meters.csv`의 사용량 |

### `meters.csv` — 검침값 (`사용량` 항목이 있을 때만)

| 호실 | 항목 | 사용량 |
|------|------|-------|
| 원유로 | 공동수도료(세대) | 11 |
| 고기나라 | 공동수도료(세대) | 61 |

- `항목`은 `costs.csv`의 항목명과 **정확히 일치**해야 합니다.

## 출력 (`bills.csv`)

호실별로 항목 금액과 합계가 한 줄씩, 맨 아래에 임차인 합계·건물 총액(검증) 행이 들어갑니다.

```
호실,평수,일반관리비,공동전기료(기본),...,공동수도료(공용),합계
원유로,44,176000,8891,...,13730,257086
...
임차인 합계,,4024000,203280,...,313914,5550670
건물 총액(검증),,4024000,203280,...,313914,5550670
```
