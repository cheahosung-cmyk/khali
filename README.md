# khali — 상가건물 관리비 정산 에이전트

**본인 소유 상가건물**의 건물주(관리소장)를 위한 **관리비 정산·청구 자동화 도구**입니다.
호실 정보·비용 항목·계량기 검침값(엑셀/CSV)만 넣으면, 항목별 배분 규칙에 따라
**임차인별 고지서를 자동 산출**하고 부가세 계산과 합계 검증까지 해 줍니다.

## 무엇을 해결하나요?

매달 반복되는 상가 관리비 청구를, 엑셀 수식과 씨름하는 대신 명령 한 줄로 끝냅니다.

- ✅ 항목별 배분 방식 자동 적용 — **면적 비례 / 호실 균등 / 사용량(검침) 비례**
- ✅ **부가세(VAT 10%) 자동 계산** — 공급가액·부가세·청구합계 구분 (세금계산서 발행 대응)
- ✅ **공실 자동 분리** — 공실 호실의 면적분은 계산해 두되 **건물주 부담**으로 표시(임차인에게 전가하지 않음)
- ✅ **원(₩) 단위 잔여까지 정확히 배분** → "항목별 합계 = 총액", "임차인+공실 = 건물총액" 1원도 안 틀림 (최대잉여법)
- ✅ 결과를 **CSV로 저장** → 엑셀에서 바로 열어 고지서/세금계산서 작성
- ✅ 별도 설치 없이 표준 라이브러리만으로 실행 (Python 3.9+)

## 빠른 시작

### 1) 샘플 입력 파일 만들기 (5층 상가 예시)

```bash
python -m src.cli sample --dir ./sample
```

`sample/` 폴더에 `units.csv`, `costs.csv`, `meters.csv` 예시 파일이 생성됩니다.

### 2) 정산 실행

```bash
python -m src.cli settle \
  --units  ./sample/units.csv \
  --costs  ./sample/costs.csv \
  --meters ./sample/meters.csv \
  --out    ./sample/bills.csv
```

요약이 화면에 출력되고, 호실별 고지서가 `bills.csv`로 저장됩니다.

> `pip install -e .` 로 설치하면 `python -m src.cli` 대신 `khali` 명령을 바로 쓸 수 있습니다.
> 부가세를 적용하지 않으려면(간이/면세 등) `--no-vat`, 세율을 바꾸려면 `--vat-rate 0.1` 옵션을 씁니다.

## 입력 파일 형식 (엑셀에서 CSV로 저장)

### `units.csv` — 호실 정보

| 층 | 호 | 계약면적 | 임차인 | 입주여부 |
|----|----|---------|--------|---------|
| 1 | 101 | 66.1 | 행복편의점 | Y |
| 3 | 301 | 99.2 |  | N |

- `입주여부`: `Y`/`N` (또는 임차/공실). 비워 두면 임차인명 유무로 자동 판단합니다.
- 공실(N)은 면적분을 계산하되 **건물주 부담**으로 분리되어 임차인 청구에서 빠집니다.

### `costs.csv` — 비용 항목 (건물 전체 월 총액, 부가세 제외 공급가액)

| 항목 | 총액 | 배분방식 |
|------|------|---------|
| 일반관리비 | 1800000 | 면적 |
| 청소비 | 600000 | 호실균등 |
| 공용전기료 | 450000 | 면적 |
| 승강기유지비 | 300000 | 면적 |
| 호실전기료 | 1650000 | 사용량 |
| 수도료 | 320000 | 사용량 |

**배분방식** 값:

| 값 | 의미 | 가중치 |
|----|------|--------|
| `면적` | 계약(전용)면적 비례 | 계약면적 |
| `호실균등` | 전 호실 균등 | 동일 |
| `사용량` | 검침 사용량 비례 | `meters.csv`의 사용량 |

### `meters.csv` — 검침값 (`사용량` 항목이 있을 때만)

| 층 | 호 | 항목 | 사용량 |
|----|----|------|-------|
| 1 | 101 | 호실전기료 | 540 |
| 1 | 101 | 수도료 | 22 |

- `항목`은 `costs.csv`의 항목명과 **정확히 일치**해야 합니다. 공실은 사용량 0이면 자동 0원.

## 출력 (`bills.csv`)

임차인 호실은 항목별 공급가액 + **공급가액·부가세·청구합계**, 공실은 건물주 부담분이 표기되고,
맨 아래에 임차인 청구 합계 / 건물주 부담(공실) / 건물 총액(검증) 행이 들어갑니다.

```
층,호,임차인,계약면적,상태,일반관리비,...,공급가액,부가세,청구합계
1,101,행복편의점,66.1,임차,218112,...,798040,79804,877844
3,301,,99.2,공실(건물주부담),327333,...,588721,-,-
...
임차인 청구 합계,,,,,...,4681279,468128,5149407
건물주 부담(공실),,,,,...,588721,-,-
건물 총액(검증),,,,,...,5270000,,
```
